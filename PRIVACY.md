# Privacy Policy

This plugin repository ships static files, a Codex skill, and a local Python CLI. It does not include built-in telemetry, analytics, or a hosted service operated by the repository maintainers.

When you run the bundled CLI:

- requests go from your environment to your configured Azure OpenAI endpoint
- authentication uses your Azure API key or Azure identity configuration
- any logging or retention is governed by your Azure, OpenAI, GitHub, and local environment policies

You are responsible for reviewing and complying with the privacy, security, and data-handling policies that apply to:

- your Azure subscription and Azure OpenAI deployment
- your Codex environment
- any systems where you store prompts, images, masks, or generated outputs

GitHub may collect standard repository, release, and website access data under GitHub's own policies when you browse or download this repository.
